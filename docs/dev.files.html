Документация по MobileCMS
Дерево каталогов

/cache

Предназначен для хранения кэша системы. В свою очередь делится на две поддиректории file_cache и smarty_templates.
Подкаталог file_cache предназначен для хранения файлов кэша системы, например, в нем хранится кэш главной страницы.

/data_files

Предназначен для хранения конфигурационных файлов системы и важных файлов данных.
Структура подкаталога:

  • каталог data_files/events необходим для хранения файлов событий различных модулей
  • каталог data_files/route_rules предназначен для хранения правил роутинга системы, напоминает rewrite_rules mod_rewrite
  • файл data_files/config.php содержит основную конфигурацию системы, такую как данные БД и адрес сайта
  • файл data_files/day.dat содержит одну цифру - текущий день, предназначен для реализации выполнения ежедневных действий в системе
  • файл data_files/forgot_message.tpl — шаблон сообщения при восстановлении пароля
  • файл data_files/reg_message.tpl — шаблон сообщения при регистрации пользователя в системе
  • файл data_files/version.dat содержит информацию о текущей версии MobileCMS

/files

Содержит файлы загрузок, аватаров, библиотеки (подкаталоги downloads, avatars, lib соответственно), а так же файлы и папки дополнительных модулей.

/install

Содержит файлы установщика.

/js

Хранит файлы JavaScript, в данный момент только JS фреймворк JQuery

/kernel

Cодержит важные файлы ядра системы. Содержимое каталога:

  • файл kernel/everyday.php скрипт, выполняющий ежедневные действия в системе, например, чистит старые файлы в каталоге временных файлов tmp
  • файл kernel/general_functions.php содержит основные функции системы, а так же аналоги некоторых функций, которых нет в ранних версиях php5. Описание функций будет приведено ниже.
  • файл kernel/ini_set.php содержит важную конфигурацию php для правильной работы MobileCMS

/libraries

Содержит основные библиотеки системы. Детальное описание библиотек смотрите в разделе краткое описание библиотек

/modules

Содержит модули системы

/tmp

Содержит временные файлы системы, которые удаляются каждые сутки

/utils

Содержит вспомогательные файлы системы

  • файл utils/captcha.php скрипт для генерации каптчи (защитное изображение)
  • файл utils/default_font.php основной шрифт системы
  • файл utils/mimes.php файл, содержащий массив mime типов
  • файл utils/time_zones.php файл, содержащий массив временных зон

/views

Содержит папки тем

  • views/admin_* темы админки
  • views/web_* web темы (в данный момент не используются)
  • папки без префиксов — основные темы сайта
