Загрузка файлов по FTP

Описание

Чтобы загружать файлы по FTP.

  • Загрузите необходимые файлы и папки в папку скрипта files/downloads/_ftp_upload
  • Зайдете в Панель управления -> Загрузки, выберите папку в которую хотели бы добавить загруженные вами по фтп файлы
  • Перейдите по ссылке "Загрузить файлы по ФТП в текущую директорию", нажмите"Обзор" и выберите необходимую папку для добавления (если папок нет, т.е. вы их не создавали в _ftp_upload, то ничего не указывайте)
  • Нажмите кнопку "Загрузить"
Файлы загружены.
