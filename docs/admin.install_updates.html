Установка обновлений

Описание

Чтобы установить обновление необходимо настроить FTP, как это сделать читайте здесь. Для установки обновлений выполните следующие действия

  • Перейдите в Панель управления -> Система -> Загрузка обновлений
  • В поле "ZIP файл" укажите путь до файла обновления на вашем компьютере или телефоне
  • Нажмите "Загрузить"
Обновление установлено.
