Настройка FTP

Описание

Настройка FTP необходима для загрузки обновлений, модулей и тем прямо из админки.

  • Перейдите в Панель управления -> Система -> Настройки FTP
  • В поле "Сервер" укажите ваш FTP сервер (обычно localhost, т.к. систем расположена на локальном сервере относительно CMS)
  • В поле "Порт" оставьте 21, либо укажите тот, который у вас
  • В поле "Пользователь" укажите имя пользователя, которое используете при входе по FTP для управления сайтом
  • В поле "Пароль" укажите пароль пользователя
  • В поле "Путь к корню MobileCMS по фтп" пропишите путь по фтп до MobileCMS начиная от корня вашего ftp аккаунта. Например, /www/site.ru
  • Нажмите "Сохранить"
